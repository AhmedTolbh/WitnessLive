# WitnessLive - Chrome Extension

An AI-powered Chrome extension that witnesses your screen in real-time to help you solve problems, debug code, and understand errors.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select the WitnessLive directory
5. The extension will appear in your browser toolbar

## Usage

1. Click the WitnessLive extension icon in your toolbar
2. Click "Launch Session" to open the application window
3. Grant screen sharing and microphone permissions when prompted
4. The AI assistant will analyze your screen and voice to help you solve technical problems

## Features

- Real-time screen sharing analysis
- Voice interaction with AI assistant
- Live transcription of conversations
- Visual markers to point to UI elements
- Google Search integration for finding documentation
- Citation tracking for AI responses

## Configuration

To use this extension, you need a Gemini API key. The extension will prompt you to configure it on first use.
