# WitnessLive - Chrome Extension

An AI-powered Chrome extension that witnesses your screen in real-time to help you solve problems, debug code, and understand errors.

## Features

- Real-time screen sharing analysis
- Voice interaction with AI assistant
- Live transcription of conversations
- Visual markers to point to UI elements
- Google Search integration for finding documentation

## Installation

### For Development
1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked"
4. Select this directory
5. Configure your Gemini API key when prompted

### For Chrome Web Store
Download the `WitnessLive.zip` file and upload it to the Chrome Web Store Developer Dashboard.

## Configuration

This extension requires a Gemini API key (free at https://aistudio.google.com/app/apikey). The extension will prompt you to configure it on first use.
